# 🌦️ WeatherNow

A modern, responsive weather application built with **HTML, CSS, and JavaScript** that provides real-time weather information and a 5-day forecast using the **OpenWeatherMap API**. The application features a clean UI, dynamic weather backgrounds, loading and error states, and a seamless user experience.

---

## 🌐 Live Demo

🔗 **Live Website:** https://ashmyth25.github.io/climate-dashboard/

---

## ✨ Features

- 🌍 Real-time weather information
- 🔍 Search weather by city
- 📅 5-Day weather forecast
- 🌡️ Current temperature
- 🌡️ Feels Like temperature
- 💧 Humidity
- 🌬️ Wind speed
- 👁️ Visibility
- ☁️ Cloud coverage
- 📈 Atmospheric pressure
- 🌅 Sunrise & Sunset timings
- 🕒 Live digital clock
- 🎨 Dynamic background based on weather conditions
- 📱 Fully responsive design
- ⚡ Loading spinner
- ❌ Error handling for invalid city names
- 🎯 Quick city search buttons
- 🌈 Smooth animations and transitions
- ♿ Keyboard support (Enter to search)

## 🚀 Installation

Clone the repository

```bash
git clone https://github.com/ashmyth25/weathernow.git
```

Navigate to the project

```bash
cd weathernow
```

Open the project

```bash
index.html
```

Or run using **Live Server** in VS Code.
